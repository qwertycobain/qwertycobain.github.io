git remote 
{% endhighlight %}

{% highlight ruby %}

{% endhighlight %}

{% highlight ruby %}

{% endhighlight %}

sobe as modificações locais para o servidor

{% highlight ruby %}

{% endhighlight %}

{% highlight ruby %}

{% endhighlight %}

{% highlight ruby %}

{% endhighlight %}

usado pra buscar e baixar conteúdo de repositório remotos e fazer atualização imediata ao repositório local
